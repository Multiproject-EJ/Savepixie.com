# Prompt: SavePixie mascot alternatives

Use case: stylized-concept

Asset type: controlled four-candidate mascot comparison sheet

Primary request: Explore four genuinely distinct SavePixie mascot silhouettes at equal scale. Each must be gender-neutral, non-human, very cute, simple, iconic, emotionally readable, and suitable for a premium savings-habit phone app used by young adults and playful adults.

Candidates:

- A — Seedling Sprite: round soft body, luminous sprout crest, tiny leaf-star wings, integrated goal-seed pouch.
- B — Petal Hood Pixie: compact round body, short petal-shaped hood, small luminous wings, simple cross-body goal-seed pouch.
- C — Star Moth Pixie: tiny central body, large symmetrical petal-moth wings forming the silhouette, glowing goal seed at its centre.
- D — Wishdrop Pixie: floating droplet/star form, tiny limbs, crescent eyes, visible mint goal-seed light inside its body.

Style/medium: polished stylised 3D or premium 2.5D character design; simple collectible forms; soft matte materials with restrained magical glow.

Composition/framing: landscape comparison board with four equal columns. Show one large neutral front pose, one tiny app-icon silhouette, and three small expressions for each candidate: curious, small success, joyful celebration.

Color palette: the same midnight indigo, deep plum, electric violet, fresh mint, warm gold, and lavender-white system across all four candidates.

Constraints: equal scale and presentation; each direction must have a genuinely different silhouette; minimal facial construction; no human child anatomy; no hair; no elf ears; no generic fairy costume; no bank, coin, crypto, casino, or classroom imagery; no resemblance to existing mascots; no UI screens; no watermark.
